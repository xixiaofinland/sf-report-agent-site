## Update the screenshot

- Replace `assets/screenshot.png` with your latest screenshot.
- Keep the same filename so the page updates without code changes.

## Update the demo video

- Open `index.html` and replace the placeholder YouTube ID:
  - Find `data-youtube-id="dQw4w9WgXcQ"`
  - Replace with your real YouTube video ID

## Publish with GitHub Pages

In GitHub:

1. Repo `Settings` -> `Pages`
2. `Source`: Deploy from a branch
3. `Branch`: `main` / `/(root)`

Your site will be served at:

`https://xixiaofinland.github.io/sf-report-agent-site/`
